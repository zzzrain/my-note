1、加载方法
document.addEventListene('DOMContentLoaded',fn);

转为数组对象：JSON.parse()
转为JSON字符：JSON.stringify()

获取元素
querySelector('#id')：返回第一个dom节点
querySelectorAll('tagName')：返回数组

获取属性：（增、删、改）
classList.add()
classList.remove()
classList.toggle()

属性长度：classList.length

自定义属性：data-name:dataset.name

3、改变函数作用域：fn.bind(this)

4、数组遍历、迭代
arr.forEach(function(item,idx,arr){
	// item：arr[i]
	// idx：索引值
});

arr.reduce(function(pre,cur,idx,arr){
	// pre：返回值
	// cur：当前值
	return pre + cur;
});

5、严格模式："use strict"
