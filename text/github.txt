git安装	https://git-for-windows.github.io

1、配置个人信息
git config --global user.name ""
git config --global user.email ""
（重新输入可以修改，--global去掉就不属于window的全局属性）

查看信息命令 git config -l

后面命令默认以git开头，忽略不写！！！
----------------------------------------------
2、本地仓库命令init
当前文件夹就会生成一个.git的隐藏目录，即本地仓库

3、git操作前需要分清三大区
工作区：除了.git以外的文件
暂存区：.git下的临时存放区域，可以理解为一个缓存文件夹——Cache
版本库：.git下的正式存放区域

4、git操作命令
log：   查看版本库文件提交信息
status：查看工作区文件变动
add + 文件名：     该文件存放到暂存区
commit -m"写备注"：暂存区文件提交到版本库

5、git首次配置

注册网站：https://github.com
          https://coding.net

首次使用需要创建ssh key
命令：ssh-keygen -t rsa -C 'xxx@xx.com'
（不用写git，写上配置的邮箱名）

在Users目录下找到.ssh里面的
id_rsa（私钥）和id_rsa.pub（公钥），远程仓库项目属性是哪个就复制哪个，然后添加到远程仓库（邮箱验证后才能添加）

检测是否连接：ssh -T git@github.com
	      ssh -T git@coding.net

6、git仓库命令

连接远程仓库：
remote add origin 远程仓库地址（项目下的http/ssh，推荐用ssh）

删除远程仓库：remote rm 远程仓库名字（默认origin）

查看远程仓库：remote -v

克隆远程仓库：clone 远程仓库地址

7、仓库配置好后，就能操作文件（本地和远程不同步就不能上传，） 
上传：push origin master
同步：pull origin master

8、版本回退：reset HEAD
   回退状态：--hard、--mixed(默认)、--soft
   慎用！！！
   撤销工作区修改：checkout -- <file>
   撤销暂存区修改：rm -- cache <file>
